📘 Calculatrice Magique – 3 Chiffres Significatifs

🎯 Description

Calculatrice scientifique avancée développée en C# sous Visual Studio 2019.Elle permet d’effectuer des calculs classiques, des conversions, la gestion des nombres complexes, et la résolution d’équations, tout en affichant les résultats avec 3 chiffres significatifs.

Ce projet est pensé comme un outil simple, rapide et fiable pour les étudiants, techniciens et passionnés de calcul scientifique.

🚀 Fonctionnalités

✨ Affichage automatique en 3 chiffres significatifs

🔢 Calculs scientifiques classiques

🧮 Gestion des nombres complexes

🧩 Résolution d’équations

🔄 Convertisseur intégré

💡 Interface simple et intuitive

🛠️ Technologies utilisées

C#

.NET Framework / Visual Studio 2019

Windows Forms (WinForms)

Solution Visual Studio : convertisseur.sln

📦 Installation

Télécharger ou cloner le dépôt :

git clone https://github.com/giragame8/calculatrice_magique_3_chiffre

Ouvrir le fichier convertisseur.sln dans Visual Studio 2019.

Restaurer les dépendances si nécessaire.

Compiler et exécuter le projet.

🕹️ Utilisation

Lancer l’application depuis Visual Studio ou via l’exécutable généré dans bin/Debug ou bin/Release.

Entrer vos valeurs dans les champs prévus.

Choisir le type de calcul ou de conversion.

Le résultat s’affiche automatiquement arrondi à 3 chiffres significatifs.
